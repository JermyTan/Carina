### Carina's BackEnd

- Backend is built with Spring Boot (OpenJDK v11.0.4 and Junit 4).
- Database is MongoDB (v3.6.3).
- Live Data is pulled from LTA DataMall with AWS Lambda
- Website hosted on AWS
